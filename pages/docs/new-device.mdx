## TTN-Sensor hinzufügen
In dieser Sektion erklären wir, wie du deinem **Projekt mit TTN-Integration** Leben einhauchen und ein paar Sensoren hinzufügen kannst. Klicke dazu in der Menüleiste oben rechts auf deinen Namen und dann auf **Meine Projekte** und wähle das Projekt deiner Wahl aus.
   <figure>
  <img style={{maxHeight: 400, border: '1px solid rgba(0, 0, 0, 0.05)'}}
    src="/images/deviceNewTTN.png"
    alt="Projektansicht"
  />
    <figcaption>
    Projektansicht für eingeloggte Nutzer:innen
  </figcaption>
  </figure>

Klicke nun auf **Sensor hinzufügen**. Dir wird nun die Eingabemaske für den neuen Sensor angezeigt. 

   <figure>
  <img style={{maxHeight: 400, border: '1px solid rgba(0, 0, 0, 0.05)'}}
    src="/images/deviceAddTTN.png"
    alt="Projektansicht"
  />
    <figcaption>
    Eingabemaske für neue Sensoren
  </figcaption>
  </figure>

Nutze als **Device-ID** die exakt gleiche Device-ID aus deiner TTN-Applikation. Den **Anzeigenamen** kannst du frei wählen.
Ein kleines Beispiel: dein Sensor heißt in deiner TTN-Konsole `node-temp-001`. Demnach lautet deine Device-ID in Stadtpuls ebenfalls `node-temp-001`. Klicke nun auf **Speichern**.

Hurray, how easy was that!
Du hast deinem Projekt einen neuen Sensor hinzugefügt.
