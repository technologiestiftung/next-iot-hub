Stadtpuls benutzt Json Web Tokens (JWT) um zu verifizieren ob ihr für eure Schreibanfragen auch autorisiert seid. 

## Neuen Token Anlegen

Logge dich in Stadtpuls.com ein und navigiere zu deinem Account indem du auf das Icon in der rechten oberen Ecke klickst oder tipst. Wähle den Reiter 'Tokens' aus. Schreibe eine Kurze bedeutsame Beschreibung zu deinem Token in das Text Feld und drücke den "Neuen Token anlegen" Knopf.
Kopiere aus dem Feld das erscheint deinen Token und sichere ihn in einem Passwort Manager oder einer sicheren Textdatei. Dieser Token erlaubt den Zugriff auf deine Sensoren. Veröffentliche ihn nicht in Blogposts und füge ihn auch nicht der Versionskontrolle an wenn du eine benutzt.

**Achtung! Wir können deinen Token nicht wiederherstellen wenn du einmal von der Seite weg navigiert bist. Du bist dafür verantwortlich ihn zu sichern.**

Sollte dein Token verloren gehen oder veröffentlich werden raten wir dir ihn zu löschen und neu anzulegen oder ihn zu rotieren.

## Existierenden Token Regenerieren

Du kannst einen Token auch neu generieren über den "Neu generieren" Knopf an jedem einzelnen Token. Auch hier öffnet sich das gleiche Feld wie beim erstellen und der neue Token wird dir einmalig angezeigt.
