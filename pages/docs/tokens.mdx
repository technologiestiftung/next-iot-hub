Stadtpuls benutzt *Json Web Tokens (JWT)*, um zu verifizieren ob du für deine Schreibanfragen autorisiert bist. 

## Neuen Token anlegen

Logge dich auf *Stadtpuls* ein und navigiere zu deinem Account, indem du auf das Icon in der rechten oberen Ecke klickst. Wähle den Reiter 'Tokens' aus. Schreibe eine kurze bedeutsame Beschreibung zu deinem Token in das Textfeld und drücke den  Button 'Neuen Token anlegen'.  
Dein persönlicher Token wird nun generiert und im Textfeld angezeigt. Dieser Token erlaubt den Zugriff auf deine Sensoren. Kopiere diesen und bewahre ihn sicher auf, am besten in einem Passwort Manager. Veröffentliche ihn auf keinen Fall, auch nicht in deinen <a href="https://stories.stadtpuls.com">Stadtpuls Stories</a> oder innerhalb eines Repositories.

*&#x26a0; Achtung! Wir können deinen Token weder einsehen noch wiederherstellen, wenn du vergessen hast, ihn zu notieren. In diesem Fall musst du dir einen neuen Token generieren.*

Sollte dein Token verloren gehen oder öffentlich werden, raten wir dir, ihn zu löschen und einen neuen anzulegen.

## Existierenden Token regenerieren

Über den "Neu generieren" Button an jedem Token kannst du einen Token auch neu generieren. Auch hier öffnet sich das gleiche Feld wie beim Erstellen und der neue Token wird dir einmalig angezeigt.
